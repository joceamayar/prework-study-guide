HTML

  • Elements
    • <html>: This is the root element that wraps the entire HTML document.
    • <head>: Contains metadata and other non-visible elements like the title, stylesheets, scripts, etc.
    • <title>: Specifies the title of the webpage displayed in the browser's title bar or tab.
    • <body>: Encloses the visible content of the webpage.
    • <h1> to <h6>: Headings of different levels, with <h1> being the highest and <h6> being the lowest.
    • <p>: Paragraph element used for text content.
    • <a>: Creates a hyperlink, allowing you to link to other web pages or resources.
    • <img>: Inserts an image into the webpage.
    • <li>: List items using <ul> or <ol> to define individual items.
    • <ul> and <ol>: Unordered and ordered lists, respectively.
    • <div>: A generic container element used for grouping and structuring other elements.
    • <span>: Inline element used for applying styles or targeting specific parts of text.
    • <table>: Creates a table with rows (<tr>) and data cells (<td> or <th>).
    • <form>: Used to create interactive forms for user input.
    • <input>: Generates an input field within a form for various types of user input (text, checkboxes, radio buttons, etc.).
    • <button>: Creates a clickable button.
    • <select> and <option>: Used to create dropdown/select menus.
  • Escaping HTML characters
    • less than symbol (<) with <
    • greater than symbol (>) with >
    • double quotes (") with "
    • single quote (’) with '
    • ampersand (&) with &
  • Coments on HTML
    • <!-- This is a comment -->

CSS,

  • CSS properties
    • color: Sets the text color. For example, color: red;
    • font-size: Sets the font size. For example, font-size: 16px;.
    • font-family: Sets the font family. For example, font-family: Arial, sans-serif;
    • background-color: Sets the background color. For example, background-color: pink;
    • padding: Sets the padding space around an element. For example, padding: 10px;.
    • margin: Sets the margin space around an element. For example, margin: 10px;.
    • border: Sets the border properties. For example, border: 1px solid black;
    • text-align: Sets the alignment of text within an element. For example, text-align: center;.
    • display: Specifies how an element should be displayed. For example, display: block; or display: inline;.
    • width and height: Sets the width and height of an element. For example, width: 200px; and height: 100px;
  • Coments on CSS
    • /* This is a comment */
