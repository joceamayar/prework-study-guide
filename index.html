HTML

  • HTML acts as a blueprint for a webpage, allowing us to define its structure and determine what elements should be included. It's like using a set of instructions to build and design a page, specifying what text, images, and other elements should be present and how they should be organized.
  • Head Elements
    • <html> This is the root element that wraps the entire HTML document.
    • <head> Contains metadata and other non-visible elements like the title, stylesheets, scripts, etc.
    • <title> Specifies the title of the webpage displayed in the browser's title bar or tab.
    • <style> Is used to embed CSS and contains the CSS rules needed to style the page.
    • <link> Defines the relationship between the current page and an external source.
    • <meta> Defines metadata for the page
    • <script> Is used to embed JavaScript on the page.
    • <noscript> Defines the content to be displayed if scripts are disabled.
  • Body Elements
    • <body> Encloses the visible content of the webpage.
    • <h1> to <h6> Headings of different levels, with <h1> being the highest and <h6> being the lowest.
    • <p> Paragraph element used for text content.
    • <a> Creates a hyperlink, allowing you to link to other web pages or resources.
    • <img> Inserts an image into the webpage.
    • <strong> and <em> elements can respectively bold or italicize a given text element.
    • <br> Creates a line of empty space between two blocks of content.
    • <li> List items using <ul> or <ol> to define individual items.
    • <ul> and <ol> Unordered and ordered lists, respectively.
    • <div> A generic container element used for grouping and structuring other elements.
  • Semantic Elements
  • Semantic elements are special tags in HTML that have names describing the purpose or meaning of the content they enclose. They are used to divide a web page into different sections.
    • <article> Used to contain an independent piece of content, such as a blog post or widget.
    • <aside> Used to contain content that often appears to one side and provides extra information about the page's topic.
    • <details> Used for additional details that the user can open or hide on demand.
    • <figcaption>element is used to define a caption for a figure or imag
    • <figure<element is used to define an image. It is often used with <figcaption>.
    • <footer< element is used to define the footer on the page.
    • <main> element is used to define the primary content and is often used with <aside> to define primary and secondary content.
    • <nav> element is used to define the area of the page with navigation links.
    • <section> element is used to group content that can stand on its own in its own section.
  • More Elements
    • <span> Inline element used for applying styles or targeting specific parts of text.
    • <table> Creates a table with rows (<tr>) and data cells (<td> or <th>).
    • <form> Used to create interactive forms for user input.
    • <input> Generates an input field within a form for various types of user input (text, checkboxes, radio buttons, etc.).
    • <button> Creates a clickable button.
    • <select> and <option> Used to create dropdown/select menus.
  • Escaping HTML characters
    • less than symbol (<) with <
    • greater than symbol (>) with >
    • double quotes (") with "
    • single quote (’) with '
    • ampersand (&) with &
  • Coments on HTML
    • <!-- This is a comment -->

CSS

  • CSS properties
    • color: Sets the text color. For example, color: red;
    • font-size: Sets the font size. For example, font-size: 16px;.
    • font-family: Sets the font family. For example, font-family: Arial, sans-serif;
    • background-color: Sets the background color. For example, background-color: pink;
    • padding: Sets the padding space around an element. For example, padding: 10px;.
    • margin: Sets the margin space around an element. For example, margin: 10px;.
    • border: Sets the border properties. For example, border: 1px solid black;
    • text-align: Sets the alignment of text within an element. For example, text-align: center;.
    • display: Specifies how an element should be displayed. For example, display: block; or display: inline;.
    • width and height: Sets the width and height of an element. For example, width: 200px; and height: 100px;
  • Coments on CSS
    • /* This is a one line comment, but the same character can be use for a multiline coment */

      • The git add command is used to tell Git that you want to include certain changes in your project's history. It prepares these changes to be saved in a future snapshot called a commit. Essentially, it's like telling Git, "Hey, remember this change, I want to keep it!"
    • git add .
      • Tells Git to include any new or modified files in that folder and all its subfolders.
    • git commit -m "commit message"
      • Saves all the staged changes as a new snapshot in Git's history. While including a description messege of the changes that are about to be committed
    • git push origin main
      • It takes all the changes you've made in your local repository after the files have been added and commited and sends them to a remote repository (GitHub) and updates it to the orign main branch.
    • git pull origin main
      • updates your local repository with the latest changes from the remote repository. It helps you stay in sync with the collaborative work and have the most up-to-date code on your computer.
